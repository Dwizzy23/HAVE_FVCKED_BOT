YT / SoundCloud / plik audio → WAV + AI stem splitter

Konwerter audio i splitter stemów AI

Wklej link YouTube/SoundCloud albo wrzuć własny plik. Panel symuluje konwersję do WAV oraz podział na vocal, drums, bass i melody. To jest gotowy frontend pod bota/aplikację — backend podpinasz pod endpoint /api/process.

Szkielet backendu

Ten frontend możesz podpiąć pod Node.js/Python. W kodzie znajdziesz funkcję sendToBackend() — teraz działa jako demo/symulacja.

POST /api/process Content-Type: multipart/form-data albo application/json { "sourceType": "link | file", "url": "https://...", "format": "wav_48", "taskType": "convert_split", "stems": ["vocal", "drums", "bass", "melody"], "mode": "studio", "note": "..." } Backend polecany: - Python FastAPI - Demucs do separacji stemów - FFmpeg do WAV/MP3/FLAC - kolejka: Redis/BullMQ/Celery - storage: lokalnie albo S3/R2
Demo frontend — używaj tylko audio, do którego masz prawa. YouTube/SoundCloud mogą mieć własne regulaminy pobierania.
